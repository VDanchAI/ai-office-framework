# Contributing

Thank you for your interest in AI Office Framework!

## How to contribute

### Found a bug?
Open an [Issue](../../issues/new/choose) using the **Bug Report** template. Specify the file and what's wrong.

### Have an idea?
Open an [Issue](../../issues/new/choose) using the **Feature Request** template. Describe what you'd like to see and why.

### Want to submit a fix?

1. Fork the repository
2. Create a branch: `git checkout -b fix/description-of-fix`
3. Make your changes
4. Make sure that:
   - Changes are synced between RU and EN versions
   - Router JSON is valid
   - Links are not broken
5. Open a Pull Request

### Translations
The framework supports Russian and English. If you'd like to add another language, open an Issue to discuss.

## What we DON'T accept

- Ready-made knowledge bases with commercial content
- Prompts copied from other sources without attribution
- Changes that break the architecture (Router → KB → Instructions)
